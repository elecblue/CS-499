# Rescue-Mate: Find The Perfect Rescue Operations Partner
This is the development repo for Rescue-Mate, an application designed for the fictional company *Grazioso Salvare* to connect their rescue operations team with suitable animal partners.
___
Install the project by cloning this directory, `npm install`, then `npm run dev` (or `bun run dev` if you're as partial as I am).
